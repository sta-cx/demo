# 情侣每日问答系统 - 30 天开发计划

## 项目概述
- **后端：** Express.js + PostgreSQL + WebSocket
- **前端：** Vue 3 + Vite
- **小程序：** WeChat Mini Program
- **特性：** AI 驱动问题、实时同步、周期性报告

## 技术栈确认

### 后端
- Node.js 18+
- Express.js 5.x
- PostgreSQL 14+
- Sequelize ORM
- Redis（缓存）
- Winston（日志）

### 前端
- Vue 3.5+
- Vite 7.x
- Pinia（状态管理）
- Axios（HTTP）
- Day.js（时间处理）

### 小程序
- WeChat Mini Program
- 原生小程序框架
- 云开发工具

5. **用户测试：** 内部测试后部署到测试环境
